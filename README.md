# 東海道五十三次 仮想旅行パッケージ

![test](https://github.com/hiroto20060205-oss/mypkg/actions/workflows/test.yml/badge.svg)

**歌川広重の作品である「東海道五十三次」に基づいて、時間経過で江戸（日本橋）～京都（三条大橋）までの道のりを出力します.**



## 動作環境
- Ubuntu 24.04.3 LTS
- **ROS2 Humble**
